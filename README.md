<li>8.動態地設計行為</li>
<li>9.行為封裝的大局觀</li>
<li>10.設計守則：多用合成，少用繼承</li>
<li>11.「策略模式」：定義了演算法家族，個別封裝起來，讓它們之間可以互相替換。此模式讓演算法的變動，不會影響到使用演算法的程式</li>
